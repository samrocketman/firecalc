Keyboard Shortcuts

  • All the Basic Equation Keys work (0-9+-*/^%<>.)
  • MC press C key
  • MR press R key
  • MS press S key
  • M+ press M key
  • pi press P key
  • () press () keys
  • Backspace press BACKSPACE key
  • CE press DELETE key
  • = press Enter (Return) key
  • EE press E key

Two Modes of Input

When the Calculator Display is focused, the keyboard shortcuts are disabled and it acts like a normal text field. This is to maximize the compatibility and give the user more control over editing their equations.
