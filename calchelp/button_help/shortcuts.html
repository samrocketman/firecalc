Button Shortcuts

  • All the Basic Equation Keys work
  • MC press C key
  • MR press R key
  • MS press S key
  • M+ press M key
  • pi press P key
  • () press () keys
  • comma press ',' key (button not listed)
  • Backspace press BACKSPACE key
  • CE press DELETE key
  • = press Enter (Return) key
  • EE press E key

I did not make a comma button because it would throw off the symmetry of the calculator and make it look ugly.Besides, the only place it is used is with the power function pow(x,y).Now that I exposed you to that small detail lets talk about power functions in the nextsection
